
# QuizInParis

#### Installation

    - telechargez une archive du git
    - extrayez l'archive et ouvrez le projet sur android studio
    - run l'application sur un mobile android ou un émulateur
    
#### Utilisation
    - entrez un login pour vous connecter à l'application avec la première lettre du prénom et le nom de famille 
      (ex : login = mjerbi et mdp : azerty)
    - vous vous connectez au MainActivity
    - pour afficher le drawer appuyez sur le boutton en haut à gauche
    - pour afficher la liste d'amis cliquez sur la fonctionnaltité Contacts du drawer
    - pour lancer le lecteur de QRcode pour la présence cliquez sur l'option Présence du Drawer
    - pour lancer le Quiz cliquez sur l'option Tester mes connaisances du drawer
    - pour se connecter au capteur cliquez sur l'option Sensor du drawer
    - pour lancer une vidéo par ip cliquez sur l'option Caméra du drawer
    
#### Quiz
    Vour lancez le jeux cliquez sur le bouton Play
    Voous serez redirigé vers une nouvelle activité qui vous affichera des questions au fur et a mesure
    Lorsque vous répondez à une question vous aurez un message qui vous inequera si vous avez répondu bon
    Vous aurez une série de 4 questions
    A la fin de cette série un mesage s'affichera  vous indiquant votre nombre de points
