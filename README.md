
# QuizInParis

### Installation

   - téléchargez une archive du git<br><br>
     extrayez l'archive et ouvrez le projet sur android studio<br><br>
     run l'application sur un mobile android ou un émulateur<br><br>
    
### Utilisation
   - entrez un login pour vous connecter à l'application avec la première lettre du prénom et le nom de famille 
      (ex : login = mjerbi et mdp : azerty)<br><br>
     vous vous connectez au MainActivity<br><br>
     pour afficher le drawer appuyez sur le boutton en haut à gauche<br><br>

### Quiz
   Vous lancez le jeux cliquez sur le bouton Play<br><br>
   Vous serez redirigé vers une nouvelle activité qui vous affichera des questions au fur et a mesure<br><br>
   Lorsque vous répondez à une question vous aurez un message qui vous indiquera si vous avez répondu bon<br><br>
   Vous aurez une série de 4 questions<br><br>
   A la fin de cette série un message s'affichera  vous indiquant votre nombre de points<br><br>
